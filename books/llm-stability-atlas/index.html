Front Matter

Stabilizing Language Model Training Without Treating Every Failure As The Same Failure

This handbook argues that “training instability” is not one problem. Public evidence splits it across residual geometry, attention-logit collapse, output-head divergence, numeric-path failures, and repetition that may be training-side or decode-side. That split matters because the standard bag of responses, lower the learning rate, clip harder, or add a repetition penalty, often acts on the wrong control surface.

What This Book Solves

Why the topic needs a handbook rather than a checklist

The hard part is not remembering that grad_clip = 1.0 exists. The hard part is deciding whether a given unstable run is failing in the residual path, in attention, in the LM head, in the precision path, or only at decode time. The later chapters are therefore organized by mechanism family, not by paper chronology.

Read Chapter 1 first if the failure signature is still unclear. Read Chapter 3 first if you already know you need a practical baseline recipe. Use the appendix when you want to check which claims are anchored in papers, docs, or partially observed implementation evidence.

Quick Conclusions

The short version

  • There is no single anti-explosion trick for decoder-only language models.
  • global grad clip = 1.0 is necessary but not sufficient; PaLM still reported roughly 20 irregular late spikes at 540B.
  • A practical default stack is Pre-LN or RMSNorm, warmup, FP32-sensitive reductions, Q/K normalization or soft-capping, z-loss, explicit [eod] packing, and repetition controls only after numeric stability is already solved.

Reading Paths

How to use the chapters

For diagnosis first

Read Chapter 1, then Chapter 2, then branch to the mechanism chapter that matches your metrics.

For recipe first

Read Chapter 3, then Chapter 4 through Chapter 7 to understand why each piece is in the stack.

For repetition issues

Read Chapter 1, then jump to Chapter 8 only after you have ruled out residual, attention, and LM-head failures.

For frontier methods

Read the core chapters first. Chapter 9 is written as a judgment page, not as a default recommendation page.
